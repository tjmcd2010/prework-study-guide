HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • ul indicates an unordered list of items that will begin with a bullet point unless otherwise defined
  • ol indicates an ordered list, be it numberical or alphabetical
  • li indicates a list item within a ul

CSS

  • Margin indicates how much space we want around the outside of an element.
  • Padding indicates how much space we want around the content inside an element.
  • ul indicates an unordered list of items that will begin with a bullet point unless otherwise defined
  • ol indicates an ordered list, be it numberical or alphabetical
  • li indicates a list item within a ul

Git Terms

  • git status - Check current branch
  • git checkout -b feature/branch-name - Creates a new branch and switches to it
  • git add -A - Adds starter code to the current working branch
  • git commit -m “add starter-code”- Commits changes to the current branch with a note explaining what the commit contains
  • git pull origin main - Ensures that local branch is in sync with base branch in github
  • git push origin feature/starter-code - Pushes changes to the remote branch

    • Git Definitions:

  • Checkout - Moves working branch to a new branch
  • -b - Creates new branch
  • -d - Deletes existing branch
  • Feature - Type of new branch.
  • Add - Adds modifications in the current working branch to the staging area.
  • -A - Adds all changes
  • Pull - Receives branch modifications
  • Origin - Source of the pull from Gihub
  • Main - Indicates the branch
  • Push - Pushes local changes to our remote Github branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
